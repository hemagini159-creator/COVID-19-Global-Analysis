# 🌍 COVID-19 Global Analysis Dashboard

## 📌 Introduction

The **COVID-19 Global Analysis Dashboard** is developed using **Tableau** to analyze and visualize the impact of the COVID-19 pandemic across different countries.
<img width="1886" height="1013" alt="image" src="https://github.com/user-attachments/assets/3ee55b2f-d07d-41a3-a5d7-e3418f4a2356" />

The dashboard compares:

* Total Population
* Confirmed Cases
* Total Deaths
* Total Recoveries

This project presents complex global health data in a clear, interactive visual format, enabling quick comparison and better decision-making.

---

## 🎯 Objective of the Dashboard

The main objectives of this dashboard are:

* To analyze total COVID-19 cases, deaths, and recoveries by country
* To compare country-wise performance during the pandemic
* To identify high-risk and low-risk countries based on death rates
* To provide a visual representation of pandemic impact relative to population
* To support data-driven insights for healthcare analysis

---

## 🛠 Tools and Technologies Used

* **Tool Used:** Tableau
* **Data Type:** Global COVID-19 dataset

### 📊 Visualization Techniques:

* Geographic Map
* Bar Charts
* Text Tables
* Interactive Filters

---

## 📊 Dashboard Components

### 1️⃣ Total Population by Country (Map Visualization)

This section displays a world map showing total population distribution by country.

**Features:**

* Each circle represents a country
* Circle size indicates population size
* Enables comparison between densely populated and smaller nations

📌 Helps understand how population size may influence COVID-19 case counts.

---

### 2️⃣ Total Deaths by Country

This section presents a country-wise breakdown of total deaths along with a **Risk Level classification**.

**Risk Categories:**

* 🔴 High Danger – High number of deaths
* 🟢 Low Danger – Lower number of deaths

**Features:**

* Bar chart showing total deaths per country
* Country filter for focused analysis

📌 Helps identify severely affected nations and compare fatality impact.

---

### 3️⃣ Total Cases – Tests – Deaths by Country

This visualization compares three major indicators:

* Total Cases
* Total Tests
* Total Deaths

**Insights Provided:**

* Testing efficiency
* Case detection rate
* Death ratio compared to confirmed cases

📌 Enables performance comparison across countries.

---

### 4️⃣ Total Recovered by Country

This section displays country-wise recovered cases.

**Purpose:**

* Measure recovery performance
* Indicate effectiveness of healthcare systems
* Compare total cases with recovery rate

📌 Countries with higher recovery numbers relative to total cases indicate better pandemic management.

---

## 🔍 Key Insights from the Dashboard

From the analysis, the following observations can be made:

* Highly populated countries tend to have higher total case numbers
* Some countries fall under the **"High Danger"** category due to high death counts
* Recovery rates vary significantly across countries
* Testing volume plays a major role in identifying confirmed cases
* The relationship between population and cases is visible in the map visualization

---

## ⚙️ Interactive Features

The dashboard includes:

* Country filter for focused analysis
* Interactive map selection
* Scrollable country comparison
* Risk level classification

These features allow users to dynamically explore and analyze the dataset.

## ✅ Conclusion

The **COVID-19 Global Analysis Dashboard** effectively presents pandemic data in a structured and interactive manner. By combining geographic visualization, comparative charts, and risk classification, the dashboard enables clear understanding of the global impact of COVID-19.

It provides meaningful insights into:

* Population impact
* Death rates
* Recovery performance
